Interior

This Construct menu command constructs the interior defined by the selected object or objects. Depending on your selection, this command may appear as Polygon Interior, Circle Interior, or Arc Interior. If the command appears as Arc Interior, it shows a submenu that allows you to choose between Arc Sector and Arc Segment.

The keyboard shortcut for Interior is Ctrl+P (Windows) or P (Mac).
