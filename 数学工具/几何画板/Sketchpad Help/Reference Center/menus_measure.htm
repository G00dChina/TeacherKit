Measure Menu

The Measure menu allows you to measure numeric properties of selected objects. The commands in the top portion of the menu measure objects’ geometric properties; the commands in the bottom portion measure analytic properties.

If you want to measure a specific property and its command is not available, make sure you have only that command’s prerequisites selected. (You may have too many or too few objects selected.)

Click the name of any command in the menu to see detailed information about that command.

 

To measure an object’s properties, select the object and choose from the available commands in the Measure menu. Sketchpad produces a measurement — a named numeric value in the proper units — as the result. When you drag or change an object that you’ve measured, the measured value changes accordingly.

To use any of these commands, you must select the appropriate objects (the prerequisites) in the sketch.
