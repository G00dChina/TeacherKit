Show All Hidden

This Display menu command displays and selects all objects previously hidden in a sketch.

You can also show hidden objects by using the Information tool or the Properties dialog box, starting from a parent or child of the hidden object. Use the Parents or Children list in the information balloon or the Object panel of the Properties dialog box to navigate to the desired hidden object. Then uncheck the Hidden checkbox.
