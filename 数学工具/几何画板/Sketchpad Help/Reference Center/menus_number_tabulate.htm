Tabulate

Selection prerequisites: One or more values, equations, or text objects that contain values, coordinates, or equations

This Number menu command creates a new table of the selected objects’ values. Sketchpad’s tables contain entries describing a set of measurements’ values at different points in time. In addition to measured values, you can tabulate parameters, calculations, measured coordinate pairs and equations, functions, and Hot Text captions.

Once you’ve created a table, you can add data to it manually or automatically, and you can remove its data in order to collect new data.

By default, new tables contain one row that dynamically tracks the current values of all tabulated measurements. Values in this row change as measured objects are dragged or animated. Turn off dynamically-changing values by choosing Edit | Properties | Table.
