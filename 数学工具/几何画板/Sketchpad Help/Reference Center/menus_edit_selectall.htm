Select All

This Edit menu command selects all objects that match the active tool in the Toolbox.

The keyboard shortcut for Select All is Ctrl+A (Windows) or A (Mac).

If a Selection Arrow, Information, or Custom tool is active, this command selects all objects in the sketch.

If another tool from the Toolbox is active (Point, Compass, Segment, Ray, Line, Polygon, Text, or Marker), all matching objects are selected. For example, if the Ray tool is active, then the command becomes Select All Rays.
