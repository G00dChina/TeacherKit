Perpendicular Line

Selection prerequisites: One of the following combinations of point(s) and straight object(s):

•One point and one straight object

•Multiple points and one straight object

•One point and multiple straight objects

This Construct menu command constructs a line through each selected point perpendicular to each selected straight object.

Selected Objects:

Resulting Construction:

One point, one straight object

One perpendicular line

Two points, one straight object

Perpendicular lines through both points

One point, two straight objects

Lines perpendicular to both straight objects
