Abscissa (x) / Polar Distance (r)

Selection prerequisites: One or more points

This Measure menu command measures the abscissa (x) or polar distance (r) of the selected points. The measurement is with respect to the marked coordinate system.

If there is no marked coordinate system, Sketchpad marks an existing coordinate system or creates a new one.

If the marked coordinate system has a square or rectangular grid form, the abscissa (x) is measured; if it has a polar grid form, the polar distance (r) is measured.

To measure both x and y (or r and θ if the grid form is polar) at the same time, hold the Shift key and choose Measure | Abscissa & Ordinate.
