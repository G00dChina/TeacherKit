# JinAn TeacherKit
 金安教师工具箱是一款服务于老师整个教学环节的小东西，希望能帮上你。

 工具箱中部分软件开发时使用了微软的DotNet框架，如果你的系统中没有支持库，在使用工具箱时可能会遇到软件打不开或者打开时报错，请自行安装一下微软的.Net支持库和VC++系列库即可解决，支持库一般在360软件管家里可以直接下载安装，以下地址亦可直达：https://share.weiyun.com/5chtZ46
