Sets pen's color to specified value

You can either set the pen color to a variable, like or a number. For example, number 0 is color red, number 70 is green, number 130 is blue, and number 170 is magenta.

Pen_color=0 at red end of rainbow, pen_color=100 at blue end of rainbow. Ranges from 0 to 200 to go around the color wheel.
