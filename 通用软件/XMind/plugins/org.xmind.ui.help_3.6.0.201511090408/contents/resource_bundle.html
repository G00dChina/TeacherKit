Resource Bundle

You can package and export your custom themes, templates, clip arts and markers to an XRB file to share them with others, in a team, or just between your different devices.

To export resource bundle
  1. Launch XMind, and click "Tools - Export Resource Bundle [Pro]"
  2. Choose the target resources in the coming dialog
  3. Click "Next", and choose the location to save the selected resource
      4. To import resource bundle
      1. Click "Tools - Import Resource Bundle"
      2. Choose the target resource bundle file, named as "**.xrb"
      3. Click "Finish"
