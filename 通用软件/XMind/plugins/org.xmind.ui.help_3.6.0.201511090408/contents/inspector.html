Inspector

Insepctor is a view to display some information and resources of current XMind file.

Including:

  1. Author Info
    • Name
    • Email
    • Organization
  2. General Info
    • Estimated size
    • Topic quantity
    • Word count
    • Revisions
    • Last modified time
    • last modified by whom
    • Create time
  3. Attachments
  4. External file links
  5. Hyperlinks
  6. Images

    7. You can click Window - Inspector from Menu to open inspector view.
