Filter

XMind's powerful filter function offers tremendous productivity improvements for large, complex maps. It allows users to make only specific portions of the map content visible, based on certain topic attributes such as markers, labels and task assignees, to facilitate faster information review and understanding.

Advanced Filter
  1. Open Advanced Filter View from "Window - Advanced Filter" on the menu.
  2. Select the marker(s) label(s) and/or task assignees that you want to use as filters.
  3. The target topic(s) with the selected marker(s) and/or label(s), will be displayed with highlights.
  4. Click "Exit" to exit filter mode.
