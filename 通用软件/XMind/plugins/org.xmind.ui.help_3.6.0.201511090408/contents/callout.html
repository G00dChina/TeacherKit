Callout

Callout is a funny way for additional text. In XMind 7, Callout is not only a shape, but also a real attachment to a topic. It also has sub-topics.

To add callout to a topic:
  1. Select a topic.
  2. Click Callout iconon the toolbar.
  3. Now you can type some text and change the callout properties in Properties window.
  4. You can also enter "Tab" to add subtopics to it.
