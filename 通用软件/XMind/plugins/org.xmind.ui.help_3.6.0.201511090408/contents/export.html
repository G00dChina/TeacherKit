Export

XMind can push our productivity to the limits. However, the mind map file is not a universal format. For sharing, we need exporting the mind map into other formats. And XMind supports a variety of file formats, like PDF, SVG, MS Excel/Word/PPT/Project, MindManager, HTML and etc.

Export to PDF(Map):
  1. Click "File - Export" from the menu.
  2. Select "PDF(Map)" in the export dialog, and click "Next" to continue.
  3. Choose the location and name the file after clicking 'Browser'.
  4. Change the settings: Page Size, Orientation.
  5. Click 'Finish' to end this process.

To Export to Mindjet MindManager Map:
  1. Choose "File > Export" on the menu.
  2. Select "Mindjet MindManager Map" in the export dialog, and click "Next" to continue
  3. Choose the location and name the file after clicking "Browser".
  4. Click "Finish" to end this process.
To Export to SVG:
  1. Choose "File > Export" on the menu
  2. Select "SVG" in the export dialog, click "Next" to continue
  3. choose the location and name the file after clicking "Browser".
  4. Click "Finish"to complete the export process.

To PDF/RTF/Microsoft Word:

  1. Click "File > Export" on the menu
  2. Select "PDF Document" in the export dialog, and click "Next" to continue.
  3. Choose the location and name the file after clicking 'Browser'.
  4. Change the settings following and check the changes in the preview dialog: 'Overview', 'Separate Overview', 'Notes', 'Floating Topics', 'Labels', 'Markers', 'Image', 'Relationships', 'Hyperlinks'.
  5. Click 'Finish'to complete the export process.

Export to CSV/Microsoft Excel:
  1. Click "File - Export" from the menu.
  2. Select "Microsoft Excel" in the export dialog, and click "Next" to continue.
  3. Choose the table layout and the contents.
  4. Choose the location and name the file after clicking "Browser".
  5. Click "Finish"to complete the export process.

Export to Microsoft PowerPoint:
  1. Click "File - Export" from the menu.
  2. Select "Mircosoft PowerPoint" in the export dialog, and click "Next" to continue.
  3. Choose the location, name the file and file format type after clicking "Browser".
  4. Change the settings, and see the changes in the preview dialog: "Overview", "Separate Overview", "Notes", "Floating Topics", "Labels", "Markers", "Image", "Relationships", "Hyperlinks".
  5. Type the "Footer text".
  6. Click "Finish"to complete the export process.

Expor to Microsoft Project
  1. Click "File - Export" from the menu.
  2. Select "Microsoft Project" in the export dialog, and click "Next" to continue.
  3. Define the exported content, and set priority mappings.
  4. Choose the location and name the file after clicking "Browser" button.
  5. Click "Finish" to complete the export process.

Note:

  • The preview is an example for your reference. The appearance of your map may vary from the examples presented here.
  • Exporting to Microsoft Word, PowerPoint, Excel and Project is available on Windows, Mac and Linux without installed Office
