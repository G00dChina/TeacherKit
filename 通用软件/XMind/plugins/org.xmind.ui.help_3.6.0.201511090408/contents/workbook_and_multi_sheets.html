Workbook and multiple sheets

Every XMind file is considered a workbook. And one workbook can maintain multiple sheets, which are separate mind maps. It's so easy to create and modify the sheet.

To create a new blank sheet:

There are several ways to create a new blank sheet.

  1. Use shortcut 'Ctrl+T'.
  2. From Sheet Context Menu:
    1. Right-click on the sheet name on the bottom of the editor viewer.
    2. Click New Blank Sheet.

  3. From Menu Commands: Insert Menu

New Sheet From Template

There are 2 ways to create new sheet from template.

  1. From Sheet Context Menu:
    1. Right-click on the sheet name on the bottom of the editor viewer.
    2. Click New Sheet From Template.
  2. From Menu Commands: Insert Menu
New Sheet From Topic

There are several ways to create a new blank sheet.

  1. Use shortcut 'Ctrl+Alt+T'.
  2. From Menu Commands: Insert Menu
  3. From Topic Context Menu:
    1. Select a topic and right click.
    2. Click New Sheet From Topic.

Modify Sheet
  1. Right-click on the sheet name on the bottom of the editor viewer.
  2. From the context menu you can make the following changes:
    • Rename this sheet.
    • Copy Sheet.
    • Duplicate Sheet.
    • Delete the current sheet from the workbook.
    • New Blank Sheets.
    • New Sheet From Template.

Note: All maps in a workbook are separate from each other. They can have different structures, wallpapers, themes, and so on.
