Import XMind Workbook

One XMind file is a workbook containing a few sheets. Under some situation, we need to merge a few files as one.

  1. Launch XMind and open the A.xmind file.
  2. Choose "Tools -- Import XMind Workbook"
  3. Select the B.xmind file and confirm.
  4. Then all sheets in B will be part of A.
